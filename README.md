# SiMSynth

SiMSynth is a small audio-sythetizer-like application.

## Installation

### Binaries
Binaries for both Windows and Linux (tested under Void Linux) are available under Releases.

### From Source
For unsupported platforms or latest features you can compile from source directly.
```bash
git clone https://github.com/silverfairies/SiMSynth
cd SiMSynth
cargo build --release
```
Cargo is required for this.
